map : key / value 형태의 데이터를 한개의 배열로 만들어서 다시 배열에 지정하는 방식

  • 선언과 초기화을 할떄는 : new Map(), new Map([['key','value'],['key', 'value'],['key', 'value']]);
  • 2차원 배열과 비슷한 구조, 기본적으로 중첩구조를 가지고 있가 떄문에 key 중복확인은 필수!
  • key를 식별자로 사용하는 get(keyName), set(keyName, value) 메서드 제공
  • key의 타입은 문자로 하는것이 원칙 => key의 타입이 달리지면 관리가 어려워진다.
  • map.set(key, value) : (key)키에 대한 값을 변경
  • map.get(key) : 키에 해당하는 value를 리턴한다.
  • map.delete(key) / has() / forEach / entries() / size
