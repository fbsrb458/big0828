변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : Number => 정수, 실수, NaN(nota number : 숫자가 아님)
  • 문자 : string => " ",' ',` ` 감싼 값들
  • 불리언 : boolean = true / false
  • 객체 : Objact => 1개 이상의 데이터가 특정 구조를 가지고 있는 형테 { 키 : 값 } ({ } : 아무것도 없는 사실도 1개의 객체로 취급)
  • 배열 : Array => [1,2,3,....] 형태로 이루어진 데이터 집합 (순서, 길이가 있음.)
  • 알수 없는 값 =null => 뭔가 있긴한테 파악이 안됨
  • 미지정된 값 : undefined => 값이 할당되지 않음
  • 데이터 타입을 확인 명열어 : typeof 변수 / 값

    • 변수의 관리

    1. 기본자료형: 값이 메모리의 주소 값이 가르키는 곳에 직접 할당한다. number, boolean, string(참조자료형이지만 사용편의성 같이 사용한다.)
    2. 참조 자료형(Reference Type) : 값이 저장되어 있는 주소를 할당한다. string,object,array,function,null,nudefined
    3. ==같다(값만 비교), != 같지 않다. ===값다(값과 타입이 값은지 비교) !== 값다(값과 타입이 같지 않은지 비교)
